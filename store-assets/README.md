Full description:

```text
Sparkline Rush is a fast neon reflex runner built for quick mobile sessions.

Swipe into a glowing lane, dodge blocks, collect sparks, and push your score higher every run. Earn chests, unlock secret ball skins, equip flashy cosmetics, stock up on shields, and use boosts when the run gets wild.

Features:
- Simple one-touch arcade gameplay
- Personal records and local score tracking
- Country/region-style launch leaderboard
- Chest rewards, secret skins, trails, and obstacle styles
- Extra lives, shields, magnets, and score boosts
- Optional rewarded ads for revives and bonus rewards

Jump in, survive longer, collect more sparks, and make the next run look better than the last.
```
