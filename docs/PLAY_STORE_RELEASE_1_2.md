# Sparkline Rush 1.2 Play Store Release

## Upload File

- File: `sparkline-rush-v1.2-release-signed.aab`
- App ID: `com.sparklinerush.game`
- Version name: `1.2`
- Version code: `3`

## What Changed

- Personal records now persist automatically at game over.
- Sparks collected during a run persist and become spendable.
- Shop purchases now use collected sparks for skins, revive tokens, and extra-life boosts.
- Extra lives can be bought and stocked, or applied during a run when hearts are missing.
- Skin cards now clearly show locked cost, equipped state, or tap-to-use state.

## Before Upload

- Internal-test the AAB on a phone.
- Confirm best score survives closing and reopening the app.
- Confirm skins, revive tokens, and extra-life boosts spend sparks correctly.
